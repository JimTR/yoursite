YourSite

This software is being developed to enable web site owners to manage a dynamic web site using a html editor and some special tags

YourSite is modular so you can create a new folder on your web site, and a default page will be created and added to your site menu. With this you can produce pages that suit you with your html editor.
NoIdeer Software have some default modules such as forums and a shop, which are easy to install to your web site

Code Ideas

YourSite uses two methods of producing variables for your dynamic templates

  • Static - these variables are added to a simple text file on your web server
  • Dynamic - these variables are created 'on the fly' by the module, there will be instruction (soon) on how to create a module
  • This allows the web site owner to add customised content to the site users
  • currently YourSite has no admin contol panel so all edits are through text files on the web server
